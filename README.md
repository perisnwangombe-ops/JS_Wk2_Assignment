# 🍎 Calorie Counter

A simple Calorie Counter web app built with HTML, CSS (Tailwind), and JavaScript for beginners.

This project helps users track their daily calorie intake by adding foods, viewing the list, seeing the total calories, and resetting for a new day.

## ✅ Project Features

- Add food items with name and calorie count
- Display all added food items for the day
- Remove individual food items
- Show total calories consumed
- Reset the counter to start a new day
- Data persists using `localStorage` (survives page refresh)
- Clean and responsive design

## 🛠️ Technologies Used

- **HTML5** - Page structure
- **Tailwind CSS** - Styling (via CDN)
- **JavaScript**  - Core logic and DOM manipulation
- **localStorage** - Persistent data storage
- **Font Awesome** - Icons

## 🚀 How to Run the Project

1. Download or clone the project folder
2. Open the `index.html` file in any web browser (Chrome, Firefox, Edge, etc.)
3. No installation or server needed!

The app works completely offline.

✨ How to Use

1. Type the food name (e.g., "Rice" or "Chicken Breast")
2. Enter the calories (e.g., 250)
3. Click **"Add Food"** button
4. See the food added to the list and total calories updated
5. Click the trash icon to remove any food item
6. Click **"Reset Day"** to clear everything and start fresh

All your data is automatically saved.
